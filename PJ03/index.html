Why this tool exists

Everything runs locally in your browser — nothing is uploaded. Supports bulk conversion, quality control, scaling, GIF optimization, RGB channel split/merge, and power-of-2 texture detection for 3D workflows.

How to use

  1. Load files — drag & drop or click. JPG, PNG, WebP, AVIF, GIF, SVG.
  2. Select a file — click in column 1. Settings apply per file; multi-select for shared settings.
  3. Adjust settings — format, quality, scale, channel config in column 2.
  4. Add Merge Node — bottom of column 1. Drop R/G/B channels, name it, click Merge.
  5. Convert — "Let it Rip" converts selected files, or all if none selected.
  6. Download — per file, all at once, or as ZIP.
