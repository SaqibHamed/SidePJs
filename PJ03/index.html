Why this tool exists

Most image conversion tools send your files to a server. This tool converts everything locally in your browser — nothing leaves your device. It supports bulk conversion, quality control, scaling, and GIF optimization, all in one place with a clear three-column workflow.

How to use

  1. Load files — drag & drop images into column 1, or click to browse. Supports JPG, PNG, WebP, AVIF, GIF, SVG.
  2. Adjust settings — choose output format, quality, scale, and max width in column 2. Click the preview to toggle original vs output.
  3. Select files — click to select individual files. Shift+click for range, Ctrl+click for multi. Settings apply to selection or all files.
  4. Convert — click the floating "Let it Rip" button. Results appear in column 3 with file size comparison.
  5. Download — download files individually or all at once as a ZIP.
